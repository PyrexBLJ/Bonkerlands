# Bonkerlands
#### Borderlands 2 & Borderlands: The Pre-Sequel SPRX for PS3

## Prerequisites
- Visual Studio 2013+
- Sony PS3 4.75+ SDK w/ Visual Studio Integration
- libpsutil

## Features
- Running PC .txt/.blcm mods
- Various quality of life features
- Basic save editing
